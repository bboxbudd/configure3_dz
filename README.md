# Учебный конфигурационный язык

Этот инструмент реализует простой конфигурационный язык с поддержкой переменных, словарей и вывода в формате YAML. Предназначен для демонстрации базовых принципов парсинга и интерпретации.

---

##  Спецификация языка

### Типы данных
- **Целые числа**: `\d+` → `42`, `0`
- **Строки**: `"..."` → `"hello"`, `"config"`
- **Словари**: `{ключ => значение, ...}`

### Имена переменных
- Должны соответствовать регулярному выражению `[_a-z]+`  
  (только строчные латинские буквы и символ подчёркивания)
- Примеры: `port`, `_tmp`, `user_name`
- Запрещены: `User`, `my-var`, `var1`
